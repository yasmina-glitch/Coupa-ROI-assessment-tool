# Coupa ROI assessment

Static GitHub + Vercel site.

## Files

- `index.html` — full calculator
- `README.md` — this file
- `vercel.json` — optional Vercel static-site config

## Design update

- No Pivot logo
- No red text
- Simple white / off-white / black / gray palette
- Inputs start empty with examples as placeholders
- ROI defaults to `0%` and NPV defaults to `$0` until all required fields are complete
- Coupa benchmark card removed
- Coupa renewal date is a normal required input field
- CTA: `Get the full ROI template`
